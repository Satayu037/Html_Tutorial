1) Image Basic src/alt

รูปภาพใน HTML ใช้แท็ก img โดยต้องมี src และควรมี alt เสมอ

<img src="images/photo.jpg" alt="ภาพตัวอย่าง">

alt สำคัญมาก: ช่วย SEO + ใช้แทนภาพเมื่อโหลดไม่ได้ + ช่วยผู้ใช้ Screen Reader

2) Resize & Aspect Ratio

ถ้ากำหนดทั้งความกว้างและสูง ให้ใช้ object-fit เพื่อไม่ให้ภาพบิด

/* ครอปภาพให้พอดีกรอบ */
.img-fixed {
  width: 260px;
  height: 160px;
  object-fit: cover;
}

(ภาพนี้ถูกบังคับขนาด 260x160 px)

3) Object fit contain fill cover

เปรียบเทียบการแสดงผลเมื่อกำหนดกรอบขนาด 800x200 px

contain (เห็นภาพครบ)

fill (ยืดจนเต็ม/ภาพเบี้ยว)

cover (ครอปเต็มพื้นที่/สวยงาม)

4) Image Link a + img

คลิกที่รูปเพื่อเปิดลิงก์ เหมาะกับ banner หรือ thumbnail

<a href="https://www.youtube.com" target="_blank">
  <img src="banner.jpg" alt="Banner">
</a>

(ลองคลิกที่รูปด้านบน)

ใช้ target="_blank" เมื่อต้องการให้ลิงก์เปิดในแท็บใหม่

5) Image Gallery grid

แสดงรูปหลายรูปด้วย Grid ให้จัดเรียงสวยและเป็นระเบียบ

.gallery { 
  display: grid; 
  grid-template-columns: repeat(auto-fill, minmax(140px, 1fr)); 
  gap: 15px; 
}

6) Responsive Images max-width: 100%

ทำให้ภาพไม่ล้นการ์ดและย่อขยายตามหน้าจออัตโนมัติ

img {
  max-width: 100%;
  height: auto;
}

7) Figure + Caption semantic

ใช้แท็ก <figure> และ <figcaption> เพื่ออธิบายภาพ (เหมาะกับบทความ)

8) ใช้รูปจากเครื่อง (Local Images) VS Code

จัดโฟลเดอร์รูปให้เป็นระเบียบ เช่น images/ แล้วอ้างอิงแบบ Relative Path

my_project/ 
 ├─ index.html 
 └─ images/ 
     ├─ photo1.jpg 
     └─ banner.png 

<img src="images/photo1.jpg" alt="My Photo">
แนะนำให้ตั้งชื่อไฟล์รูปเป็นภาษาอังกฤษตัวเล็ก ไม่มีช่องว่าง (ใช้ - แทน) เช่น my-photo-1.jpg
