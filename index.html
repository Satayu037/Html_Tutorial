HTML Document Structure

โครงสร้างพื้นฐานของไฟล์ HTML ที่ถูกต้อง

คำอธิบาย:
    <!DOCTYPE html>: ระบุว่าเป็นเอกสาร HTML5
    <html>: เป็นราก (Root) ของเอกสาร HTML ทั้งหมด
    <head>: ส่วนที่ใช้เก็บข้อมูลเมตา เช่น <title>, CSS, Encoding
    <body>: ส่วนเนื้อหาที่จะแสดงผลบนหน้าเว็บ

องค์ประกอบสำคัญ:
    - แท็ก (Tags): <h1>...</h1>
    - แอตทริบิวต์ (Attributes): <img src="..." alt="...">

Introduction to HTML

- HTML stands for Hyper Text Markup Language
- HTML is the standard markup language for creating Web pages
- HTML describes the structure of a Web page
- HTML elements tell the browser how to display the content
- HTML is NOT a programming language, but a markup language

Text Formatting View Workshop

การปรับแต่งรูปแบบตัวอักษร

<b>   : Bold text (ตัวหนา)
<i>   : Italic text (ตัวเอียง)
<u>   : Underline text (ขีดเส้นใต้)
<strong> : Important text (ตัวหนาเชิงความหมาย)
<em>  : Emphasized text (ตัวเอียงเชิงความหมาย)
<mark> : Highlighted text (ไฮไลท์สีเหลือง)
<del> : Deleted text (ขีดฆ่า)
<sup> : Superscript (ตัวยก)
<sub> : Subscript (ตัวห้อย)

Creating Links

การสร้างจุดเชื่อมโยง (Hyperlinks)

1. ลิงก์ไปเว็บอื่น:
   <a href="https://www.google.com" target="_blank">Go to Google</a>
   (target="_blank" คือเปิดหน้าต่างใหม่)

2. ลิงก์ภายในหน้าเดียวกัน (Anchor):
   <a href="#section-id">Jump to Section</a>

3. ลิงก์อีเมล:
   <a href="mailto:support@example.com">Email Us</a>

Lists (Ordered & Unordered)

Ordered List (รายการแบบมีลำดับ - ตัวเลข):

Adding Images

Syntax พื้นฐาน:
<img src="image.jpg" alt="คำอธิบายรูปภาพ" width="300">

Attribute ที่ควรรู้:
- src: ที่อยู่ของไฟล์รูป (URL หรือ Path)
- alt: ข้อความที่จะแสดงเมื่อโหลดรูปไม่ได้ (สำคัญต่อ SEO)
- width/height: กำหนดขนาด

HTML Forms WS 1 WS 2

1. โครงสร้างพื้นฐาน
   <form action="/submit" method="POST"> ... </form>

2. Input Types ยอดนิยม
   <input type="text" placeholder="Username">

   <input type="email">
   <input type="date">
   <input type="radio" name="gender"> Male
   <input type="checkbox"> I agree

3. ปุ่ม (Buttons)
   <button type="submit">Send</button>
   <button type="reset">Clear</button>

4. Dropdown
   <select>
      <option>Option 1</option>
   </select>
